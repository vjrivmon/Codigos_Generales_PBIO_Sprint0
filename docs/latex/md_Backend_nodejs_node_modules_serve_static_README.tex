\hypertarget{md_Backend_nodejs_node_modules_serve_static_README_autotoc_md2978}{}\doxysubsection{serve\+Static(root, options)}\label{md_Backend_nodejs_node_modules_serve_static_README_autotoc_md2978}
Create a new middleware function to serve files from within a given root directory. The file to serve will be determined by combining {\ttfamily req.\+url} with the provided root directory. When a file is not found, instead of sending a 404 response, this module will instead call {\ttfamily next()} to move on to the next middleware, allowing for stacking and fall-\/backs.\hypertarget{md_Backend_nodejs_node_modules_serve_static_README_autotoc_md2979}{}\doxysubsubsection{Options}\label{md_Backend_nodejs_node_modules_serve_static_README_autotoc_md2979}
\hypertarget{md_Backend_nodejs_node_modules_serve_static_README_autotoc_md2980}{}\doxyparagraph{accept\+Ranges}\label{md_Backend_nodejs_node_modules_serve_static_README_autotoc_md2980}
Enable or disable accepting ranged requests, defaults to true. Disabling this will not send {\ttfamily Accept-\/\+Ranges} and ignore the contents of the {\ttfamily Range} request header.\hypertarget{md_Backend_nodejs_node_modules_serve_static_README_autotoc_md2981}{}\doxyparagraph{cache\+Control}\label{md_Backend_nodejs_node_modules_serve_static_README_autotoc_md2981}
Enable or disable setting {\ttfamily Cache-\/\+Control} response header, defaults to true. Disabling this will ignore the {\ttfamily immutable} and {\ttfamily max\+Age} options.\hypertarget{md_Backend_nodejs_node_modules_serve_static_README_autotoc_md2982}{}\doxyparagraph{dotfiles}\label{md_Backend_nodejs_node_modules_serve_static_README_autotoc_md2982}
Set how \char`\"{}dotfiles\char`\"{} are treated when encountered. A dotfile is a file or directory that begins with a dot (\char`\"{}.\char`\"{}). Note this check is done on the path itself without checking if the path actually exists on the disk. If {\ttfamily root} is specified, only the dotfiles above the root are checked (i.\+e. the root itself can be within a dotfile when set to \char`\"{}deny\char`\"{}).


\begin{DoxyItemize}
\item `\textquotesingle{}allow'{\ttfamily No special treatment for dotfiles. -\/}\textquotesingle{}deny\textquotesingle{}{\ttfamily Deny a request for a dotfile and 403/}next(){\ttfamily . -\/}\textquotesingle{}ignore\textquotesingle{}{\ttfamily Pretend like the dotfile does not exist and 404/}next()\`{}.
\end{DoxyItemize}

The default value is similar to `\textquotesingle{}ignore'\`{}, with the exception that this default will not ignore the files within a directory that begins with a dot.\hypertarget{md_Backend_nodejs_node_modules_serve_static_README_autotoc_md2983}{}\doxyparagraph{etag}\label{md_Backend_nodejs_node_modules_serve_static_README_autotoc_md2983}
Enable or disable etag generation, defaults to true.\hypertarget{md_Backend_nodejs_node_modules_serve_static_README_autotoc_md2984}{}\doxyparagraph{extensions}\label{md_Backend_nodejs_node_modules_serve_static_README_autotoc_md2984}
Set file extension fallbacks. When set, if a file is not found, the given extensions will be added to the file name and search for. The first that exists will be served. Example\+: `\mbox{[}\textquotesingle{}html', \textquotesingle{}htm\textquotesingle{}\mbox{]}\`{}.

The default value is {\ttfamily false}.\hypertarget{md_Backend_nodejs_node_modules_serve_static_README_autotoc_md2985}{}\doxyparagraph{fallthrough}\label{md_Backend_nodejs_node_modules_serve_static_README_autotoc_md2985}
Set the middleware to have client errors fall-\/through as just unhandled requests, otherwise forward a client error. The difference is that client errors like a bad request or a request to a non-\/existent file will cause this middleware to simply {\ttfamily next()} to your next middleware when this value is {\ttfamily true}. When this value is {\ttfamily false}, these errors (even 404s), will invoke {\ttfamily next(err)}.

Typically {\ttfamily true} is desired such that multiple physical directories can be mapped to the same web address or for routes to fill in non-\/existent files.

The value {\ttfamily false} can be used if this middleware is mounted at a path that is designed to be strictly a single file system directory, which allows for short-\/circuiting 404s for less overhead. This middleware will also reply to all methods.

The default value is {\ttfamily true}.\hypertarget{md_Backend_nodejs_node_modules_serve_static_README_autotoc_md2986}{}\doxyparagraph{immutable}\label{md_Backend_nodejs_node_modules_serve_static_README_autotoc_md2986}
Enable or disable the {\ttfamily immutable} directive in the {\ttfamily Cache-\/\+Control} response header, defaults to {\ttfamily false}. If set to {\ttfamily true}, the {\ttfamily max\+Age} option should also be specified to enable caching. The {\ttfamily immutable} directive will prevent supported clients from making conditional requests during the life of the {\ttfamily max\+Age} option to check if the file has changed.\hypertarget{md_Backend_nodejs_node_modules_serve_static_README_autotoc_md2987}{}\doxyparagraph{index}\label{md_Backend_nodejs_node_modules_serve_static_README_autotoc_md2987}
By default this module will send \char`\"{}index.\+html\char`\"{} files in response to a request on a directory. To disable this set {\ttfamily false} or to supply a new index pass a string or an array in preferred order.\hypertarget{md_Backend_nodejs_node_modules_serve_static_README_autotoc_md2988}{}\doxyparagraph{last\+Modified}\label{md_Backend_nodejs_node_modules_serve_static_README_autotoc_md2988}
Enable or disable {\ttfamily Last-\/\+Modified} header, defaults to true. Uses the file system\textquotesingle{}s last modified value.\hypertarget{md_Backend_nodejs_node_modules_serve_static_README_autotoc_md2989}{}\doxyparagraph{max\+Age}\label{md_Backend_nodejs_node_modules_serve_static_README_autotoc_md2989}
Provide a max-\/age in milliseconds for http caching, defaults to 0. This can also be a string accepted by the \href{https://www.npmjs.org/package/ms\#readme}{\texttt{ ms}} module.\hypertarget{md_Backend_nodejs_node_modules_serve_static_README_autotoc_md2990}{}\doxyparagraph{redirect}\label{md_Backend_nodejs_node_modules_serve_static_README_autotoc_md2990}
Redirect to trailing \char`\"{}/\char`\"{} when the pathname is a dir. Defaults to {\ttfamily true}.\hypertarget{md_Backend_nodejs_node_modules_serve_static_README_autotoc_md2991}{}\doxyparagraph{set\+Headers}\label{md_Backend_nodejs_node_modules_serve_static_README_autotoc_md2991}
