{\bfseries{This library validates and sanitizes strings only.}}

If you\textquotesingle{}re not sure if your input is a string, coerce it using `input + '\textquotesingle{}\`{}. Passing anything other than a string will result in an error.\hypertarget{md_Backend_nodejs_node_modules_validator_README_autotoc_md3394}{}\doxysection{Installation and Usage}\label{md_Backend_nodejs_node_modules_validator_README_autotoc_md3394}
